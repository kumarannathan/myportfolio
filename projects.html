Stock Simulation

A stock market simulation wrapped as a war simulator. The program received as input a list of orders that specified whether the trader intended to buy or sell a specific stock. As output, the program would simulate a stock market by printing when a certain stock is bought or sold when it is profitable to do so. I also implemented a “streaming algorithm” to calculate the optimal point at which to buy and later sell commodities to achieve the highest profit margin.

    main components:

  • C++
  • Runtime Efficency
  • Priority Queues
  • Running Median

Pokemon Trainer Mini Game

This project involved implementing Minimum Spanning Tree (MST) algorithms, such as Prim's and Kruskal's, and determining their efficiency for various scenarios. I also implemented a Branch and Bound algorithm for the Traveling Salesman Problem (TSP), along with developing fast bounding algorithms and exploring heuristic approaches for achieving near-optimal solutions.

Through this project, I gained practical experience in algorithm implementation and optimization, while utilizing graph visualization tools for debugging.

    main components:

  • Prims Algorithm
  • Kruskals Algorithm
  • MST/TSP Heuristics
